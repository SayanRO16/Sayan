# 🛒 High-Performance Vanilla JS Cart Drawer Engine

A zero-dependency, ultra-lightweight **E-Commerce Cart Drawer Engine** written in modern ES6 JavaScript and CSS Grid/Flexbox. Designed for integration into any E-commerce platform (Shopify, Custom Node backends, WooCommerce).

## 🏎️ Features

- 3.2 KB gzipped bundle size.
- Accessible ARIA dialog controls (Keyboard tab focus traps).
- Built-in pub-sub event listener (`cart:updated`, `cart:opened`, `cart:error`).
