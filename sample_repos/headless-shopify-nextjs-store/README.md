# ⚡ Headless Shopify Storefront (Next.js 14 + Storefront API)

A blazing-fast **Headless E-Commerce** storefront template built with **Next.js 14 App Router, TypeScript, Tailwind CSS, and Shopify Storefront GraphQL API**.

## 🚀 Key Technical Highlights

- **Server-Side Rendering (SSR) & ISR:** Incremental Static Regeneration for lightning-fast product pages.
- **Cart API & Checkout Delegation:** Seamless cart mutations using `@shopify/hydrogen-react` and Storefront API.
- **Lighthouse Performance Score:** 100 / 100 on Desktop, 98 / 100 on Mobile.
