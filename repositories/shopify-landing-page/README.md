## 📖 Description

Designed specifically for DTC (Direct-To-Consumer) brands running paid ad campaigns (Meta, Google, TikTok Ads). This repository provides a self-contained, customizable landing page section complete with video backgrounds, countdown timers, feature highlight badges, customer review callouts, and instant buy actions.

---

## ✨ Features

- 🎯 **Conversion Rate Optimized (CRO):** Built according to top-performing e-commerce landing page layouts.
- ⏱️ **Urgency & Scarcity Countdown Timer:** Integrated dynamic promotional timer.
- 📱 **Mobile First Grid:** Sub-second responsive layout engineered for mobile ad traffic.
- 🎨 **Full Theme Editor Control:** Customizable colors, headings, call-to-action buttons, and trust badges via Shopify Customizer.

## 🚀 Installation

1. Copy `sections/landing-hero-builder.liquid` into your theme's `sections/` directory.
2. Copy `assets/landing.css` into your theme's `assets/` directory.
3. Open Shopify Customizer -> Add Section -> **Landing Page Hero Builder**.

---

## 💻 Usage Example

Add the section directly onto any custom page template or product page in OS 2.0:

```json
{
  "type": "landing-hero-builder",
  "settings": {
    "heading": "Experience Next-Gen Comfort",
    "cta_label": "Claim Special Offer"
  }
}
```
